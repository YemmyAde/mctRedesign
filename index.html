Web Design

Web design refers to the design of websites (Page) that are displayed on the internet, a web designer works on the appearance, layout, and, in some cases, content of a website. Appearance, for instance, relates to the colours, font, and images used. Layout refers to how information is structured and categorized. A good web design is easy to use, aesthetically pleasing and suits the user group and brand of the website. In this course, we will learn in a fun and easy way.
